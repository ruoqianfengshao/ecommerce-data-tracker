# 电商选品平台 - 功能开发清单

## 已实现功能 ✅
- 数据展示（日期选择、产品卡片）
- 品类过滤
- 分类图表
- 平台统计
- 懒加载（Load More）

## 待实现功能 ❌

### P0 - 核心功能
1. **搜索功能** - 按产品名称、标签搜索
2. **排序功能** - 按利润率、需求、评分等排序
3. **详情弹窗** - 点击产品查看详细信息
4. **数据导出** - 导出 CSV/Excel
5. **收藏功能** - 标记感兴趣的产品

### P1 - 增强功能
6. **趋势图表** - 显示历史数据趋势
7. **对比功能** - 多个产品横向对比
8. **筛选器** - 利润率区间、需求区间等高级筛选
9. **暗色模式** - Dark Mode 支持
10. **分享功能** - 生成分享链接

### P2 - 优化功能
11. **性能优化** - 虚拟滚动、缓存
12. **离线支持** - Service Worker
13. **多语言** - 中英文切换
14. **数据分析报告** - 生成周报/月报

## 测试需求
- Unit Tests (Vitest)
- Component Tests (Vue Test Utils)
- E2E Tests (Playwright)
